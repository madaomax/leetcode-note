You are given an `m x n` grid `rooms` initialized with these three possible values.

- `-1` A wall or an obstacle.
- `0` A gate.
- `INF` Infinity means an empty room. We use the value `231 - 1 = 2147483647` to represent `INF` as you may assume that the distance to a gate is less than `2147483647`.

Fill each empty room with the distance to _its nearest gate_. If it is impossible to reach a gate, it should be filled with `INF`.

**Example 1:**

![](https://assets.leetcode.com/uploads/2021/01/03/grid.jpg)

**Input:** rooms = [[2147483647,-1,0,2147483647],[2147483647,2147483647,2147483647,-1],[2147483647,-1,2147483647,-1],[0,-1,2147483647,2147483647]]
**Output:** [[3,-1,0,1],[2,2,1,-1],[1,-1,2,-1],[0,-1,3,4]]

**Example 2:**

**Input:** rooms = [[-1]]
**Output:** [[-1]]

**Constraints:**

- `m == rooms.length`
- `n == rooms[i].length`
- `1 <= m, n <= 250`
- `rooms[i][j]` is `-1`, `0`, or `231 - 1`.


# Solution 1: multi-source BFS
```python
class Solution:
    def wallsAndGates(self, rooms: List[List[int]]) -> None:
        """
        Do not return anything, modify rooms in-place instead.

        Args:
            m*n grid `rooms` init'd w/
                - -1 = A wall / an obstacle
                - 0 = a gate
                - inf(2147483647) means an empty room. 
        Fill each empty room with the distance to its nearest gate.
        if impossible, it should be filled with INF. 

        Assumptions:
        -  you may assume that the distance to a gate is less than 2147483647.
        - 1 <= m, n <= 250
        - rooms[i][j] is -1, 0, or 231 - 1.

        Pattern:
        find nearest path in unweighted graph: BFS
        multiple starting points: multi-source BFS

        """
        m = len(rooms)
        n = len(rooms[0])
        queue = deque()
        # start a multi-source BFS from the gates and mark the empty rooms with
        # "minutes"/ steps traveled so far. 
        for i in range(m):
            for j in range(n):
                if rooms[i][j] == 0:
                    queue.append((i,j))

        if len(queue) == 0:
            return rooms
        
        directions = [(1,0),(-1,0),(0,1),(0,-1)]

        while queue:
            i, j = queue.popleft()
            for direction in directions:
                di, dj = direction
                ni, nj = i+di, j+dj
                if 0<=ni<m and 0<=nj<n and rooms[ni][nj] == 2147483647:
                    # == inf means 1. not a wall, 2. it is an unvisited, empty room
                    # when it was first visited, it must be the shortest distance.
                    rooms[ni][nj] = rooms[i][j] + 1
                    queue.append((ni,nj))
        return rooms
```

Each cell in grid is visited at most once.
Space complexity: in-place modification, no extra visited matrix. O(mn) because of BFS queue. 
Time complexity O(mn)
