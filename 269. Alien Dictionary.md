There is a new alien language that uses the English alphabet. However, the order of the letters is unknown to you.

You are given a list of strings `words` from the alien language's dictionary. Now it is claimed that the strings in `words` are **sorted lexicographically** by the rules of this new language.

If this claim is incorrect, and the given arrangement of string in `words` cannot correspond to any order of letters, return `"".`

Otherwise, return _a string of the unique letters in the new alien language sorted in **lexicographically increasing order** by the new language's rules__._ If there are multiple solutions, return _**any of them**_.

**Example 1:**

**Input:** words = ["wrt","wrf","er","ett","rftt"]
**Output:** "wertf"

**Example 2:**

**Input:** words = ["z","x"]
**Output:** "zx"

**Example 3:**

**Input:** words = ["z","x","z"]
**Output:** ""
**Explanation:** The order is invalid, so return `""`.

**Constraints:**

- `1 <= words.length <= 100`
- `1 <= words[i].length <= 100`
- `words[i]` consists of only lowercase English letters.


# Solution 1: build adjacency list + topological sort
```python
class Solution:
    def alienOrder(self, words: List[str]) -> str:
        """
        Summary:
        
        A new alien language that uses the English alphabet. The order of the letters is unknown to you.
        The strings in `words` are sorted lexicographically by the rules of this new language.

        A string a is lexicographically smaller than a string b if in the first position where a and b differ, string a has a letter that appears earlier in the alien language than the corresponding letter in b. If the first min(a.length, b.length) characters do not differ, then the shorter string is the lexicographically smaller one.

        Args:
            A list of strings `words` from the alien language's dictionary. 
            
        Returns:
            If the claim is incorrect, i.e. the strings in words are not sorted lexicographically, and the 
            given arrangement of string in `words` cannot correspond to any order of letters, return ""

            o/w, return a string of the unique letters in the new alien language sorted in lexicographically
            increasing order by the new language rules. If there are multiple solutions, return any of them.

        Assumptions:
        - 1<= words.length <= 100
        - 1 <= words[i].length <= 100
        - words[i] consists of only lowercase English letters.
        """

        # define: if there is a directed edge from letter A to B, then it means A is lexicographically smaller
        # than B.

        # If such graph that we built is not a DAG, i.e. there is a cycle, then we return "".
        # If such graph is a DAG, return its topological sort. 

        # Step 0: Step 1 and above works with lenwords>=2
        if len(words) == 1:
            return "".join(set(words[0]))

        # Step 1: make an adjacency list.
        adj = defaultdict(set)
        indegree = defaultdict(int)
        alphabet = set()
        for i in range(len(words)-1):
            # compare words[i] and words[i+1]
            short_len = min(len(words[i]), len(words[i+1]))
            long_len = max(len(words[i]), len(words[i+1]))
            found = False
            for j in range(short_len):
                if words[i][j] != words[i+1][j] and not found:
                    smaller_letter = words[i][j]
                    larger_letter = words[i+1][j]
                    found = True
                    if larger_letter not in adj[smaller_letter]: # O(1) for a set
                        adj[smaller_letter].add(larger_letter)
                        indegree[larger_letter] += 1
                alphabet.add(words[i][j])
                alphabet.add(words[i+1][j])
            # add remaining to alphabet:
            for j in range(short_len, long_len):
                if len(words[i]) < len(words[i+1]):
                    alphabet.add(words[i+1][j])
                else:
                    if not found:
                        return ""
                    alphabet.add(words[i][j])
        

        # then apply Kahn's algorithm
        ans = ""
        queue = deque()
        for char in alphabet:
            if indegree[char] == 0:
                queue.append(char)

        while queue:
            cur = queue.popleft()
            ans += cur

            for larger_char in adj[cur]:
                indegree[larger_char] -= 1
                if indegree[larger_char] == 0:
                    queue.append(larger_char)
        
        if len(ans) == len(alphabet):
            return ans
        else:
            return ""
        
```

Runtime: O(C) where C is the number of characters touched.
Space complexity: O(N) where C is number of alphabet


Same solution but with better style and efficient string building to avoid re-creating new strings to `ans`:

class Solution:
    def alienOrder(self, words: List[str]) -> str:
        """
        Summary:
        
        A new alien language that uses the English alphabet. The order of the letters is unknown to you.
        The strings in `words` are sorted lexicographically by the rules of this new language.

        A string a is lexicographically smaller than a string b if in the first position where a and b differ, string a has a letter that appears earlier in the alien language than the corresponding letter in b. If the first min(a.length, b.length) characters do not differ, then the shorter string is the lexicographically smaller one.

        Args:
            A list of strings `words` from the alien language's dictionary. 
            
        Returns:
            If the claim is incorrect, i.e. the strings in words are not sorted lexicographically, and the 
            given arrangement of string in `words` cannot correspond to any order of letters, return ""

            o/w, return a string of the unique letters in the new alien language sorted in lexicographically
            increasing order by the new language rules. If there are multiple solutions, return any of them.

        Assumptions:
        - 1<= words.length <= 100
        - 1 <= words[i].length <= 100
        - words[i] consists of only lowercase English letters.
        """

        # define: if there is a directed edge from letter A to B, then it means A is lexicographically smaller
        # than B.

        # If such graph that we built is not a DAG, i.e. there is a cycle, then we return "".
        # If such graph is a DAG, return its topological sort. 

        # Step 0: collect unique alphabet:
        alphabet = set()
        for word in words:
            for char in word:
                alphabet.add(char)

        # Step 1: make an adjacency list.
        adj = defaultdict(set)
        indegree = defaultdict(int)
        for i in range(len(words)-1):
            # compare words[i] and words[i+1]
            min_len = min(len(words[i]), len(words[i+1]))

            found = False # found the lexicographically comparable char pair
            for j in range(min_len):
                if words[i][j] != words[i+1][j]:
                    smaller_letter = words[i][j]
                    larger_letter = words[i+1][j]
                    if larger_letter not in adj[smaller_letter]: # O(1) for a set
                        adj[smaller_letter].add(larger_letter)
                        indegree[larger_letter] += 1
                    found = True
                    break
            if len(words[i]) > len(words[i+1]) and not found:
                return ""
        
        # then apply Kahn's algorithm
        ans = []
        queue = deque()
        for char in alphabet:
            if indegree[char] == 0:
                queue.append(char)

        while queue:
            cur = queue.popleft()
            ans.append(cur)

            for larger_char in adj[cur]:
                indegree[larger_char] -= 1
                if indegree[larger_char] == 0:
                    queue.append(larger_char)
        
        return "".join(ans) if len(ans) == len(alphabet) else ""
