# [30. Substring with Concatenation of All Words](https://leetcode.com/problems/substring-with-concatenation-of-all-words/)

**Hard**

You are given a string `s` and an array of strings `words`. All the strings of `words` are of **the same length**.

A **concatenated string** is a string that exactly contains all the strings of any permutation of `words` concatenated.

- For example, if `words = ["ab","cd","ef"]`, then `"abcdef"`, `"abefcd"`, `"cdabef"`, `"cdefab"`, `"efabcd"`, and `"efcdab"` are all concatenated strings. `"acdbef"` is not a concatenated string because it is not the concatenation of any permutation of `words`.

Return an array of *the starting indices* of all the concatenated substrings in `s`. You can return the answer in **any order**.



**Sliding window**

The Key Insight

Since every word has the same length `wlen`, the concatenated substring has a fixed length of `wlen * len(words)`. So you're looking for a window of exactly that size where the word-frequency inside matches the frequency of `words`.

Think of it as **sliding window on words, not characters**.



 so you can chunk `s` into `wlen`-sized pieces and compare whole words at once.

The catch: the chunking depends on your starting offset. `"barfoo..."` chunks differently starting at index 0 vs 1 vs 2. So you run this for offsets `0, 1, ..., wlen-1`.





```python
class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
        """
        Summary:
        A concatenated str is a str that exactly contains all the 
        strs of any permutation of `words` concatenated. 

        Args:
            s: str
            words: an arr of strings. All the strs of words are of
            the same length

        Return:
            an array of the starting indices of all the concatenated
            substrings in `s`.
            You can return the answer in any order
        e.g. 
        s = "barfoothefoobarman", words = ["foo","bar"]
        Output: [0,9]
        left = 0
        right = 0

        Input: s = "wordgoodgoodgoodbestword", words = ["word","good","best","word"]

        Output: []
        """

        # sliding window
        # fixed wlen, result concatenation is fixed: wlen * len(words)
        wlen = len(words[0])
        n = len(words)
        # sliding window ptrs
        left = 0
        right = 0
        # k: word, v: freq in `words`. Result substr needs to match 
        target = Counter(words) # O(n)
        result = []

        # The result str is always `n * wlen`
        # so only need to loop `s` through possible offsets
        for offset in range(wlen):
            # start substr at different offset
            left = offset
            # how many valid words in window. ultimatly need to == n
            count = 0
            # examine if window == target
            window = Counter()
            
            # examine each word. step `right` one word at a time.
            # set max at the start of last possible word
            # because we use right to start the word
            for right in range(offset, len(s)-wlen+1, wlen):
                # current word in s
                word = s[right:right + wlen]
                if word in target:  # search for key in hash table, O(1)
                    count += 1                                
                    window[word] += 1

                    # if we have too many of this word, shrink window.
                    while window[word] > target[word]:
                        # remove leftmost word until this window is valid
                        left_word = s[left:left+wlen]
                        window[left_word] -= 1
                        count -= 1
                        left += wlen
                    
                    # if full match, `left` is valid result.
                    if count == n:
                        result.append(left)
                else:
                    # word not in target, window needs a reset
                    window.clear()
                    count = 0
                    left = right + wlen # set at next value of right
        return result




```
