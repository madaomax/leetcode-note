Given a string `s` and an integer `k`, return _the maximum number of vowel letters in any substring of_ `s` _with length_ `k`.

**Vowel letters** in English are `'a'`, `'e'`, `'i'`, `'o'`, and `'u'`.

**Example 1:**

**Input:** s = "abciiidef", k = 3
**Output:** 3
**Explanation:** The substring "iii" contains 3 vowel letters.

**Example 2:**

**Input:** s = "aeiou", k = 2
**Output:** 2
**Explanation:** Any substring of length 2 contains 2 vowels.

**Example 3:**

**Input:** s = "leetcode", k = 3
**Output:** 2
**Explanation:** "lee", "eet" and "ode" contain 2 vowels.

**Constraints:**

- `1 <= s.length <= 105`
- `s` consists of lowercase English letters.
- `1 <= k <= s.length`



# Solution 1: fixed-window sliding
```python

class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        """
        Summary:
            vowel letters in English are  'a', 'e', 'i', 'o', and 'u'.
        Args:
            string `s`
            int `k`
        Returns:
            maximum number of vowel letters in any substring of `s` *with* length `k`
        """
        max_num_vowel = 0
        # M0: keep trying sliding window?
        vowels = {'a', 'e', 'i', 'o', 'u'} # O(1) lookup
        left = 0
        # right = 0
        cur_num_vowel = 0
        # cur_length = right - left
        # if right - left == k
        for right in range(len(s)):
            if s[right] in vowels:
                cur_num_vowel += 1
            substr_len = right - left + 1
            # the window is valid only if the window is consecutive, and length is k.
            if substr_len > k: 
                # right - left + 1 > k
                if s[left] in vowels:
                    cur_num_vowel -= 1
                left += 1
            # we can update every loop as if the window len < k,
            # because the nature of the problem, it is guaranteed that
            # when window expands to k, cur_num_vol will either remain
            # the same, or increase.
            max_num_vowel = max(max_num_vowel, cur_num_vowel)
            
        return max_num_vowel```
