Given an array of integers `nums`, calculate the **pivot index** of this array.

The **pivot index** is the index where the sum of all the numbers **strictly** to the left of the index is equal to the sum of all the numbers **strictly** to the index's right.

If the index is on the left edge of the array, then the left sum is `0` because there are no elements to the left. This also applies to the right edge of the array.

Return _the **leftmost pivot index**_. If no such index exists, return `-1`.

**Example 1:**

**Input:** nums = [1,7,3,6,5,6]
**Output:** 3
**Explanation:**
The pivot index is 3.
Left sum = nums[0] + nums[1] + nums[2] = 1 + 7 + 3 = 11
Right sum = nums[4] + nums[5] = 5 + 6 = 11

**Example 2:**

**Input:** nums = [1,2,3]
**Output:** -1
**Explanation:**
There is no index that satisfies the conditions in the problem statement.

**Example 3:**

**Input:** nums = [2,1,-1]
**Output:** 0
**Explanation:**
The pivot index is 0.
Left sum = 0 (no elements to the left of index 0)
Right sum = nums[1] + nums[2] = 1 + -1 = 0

**Constraints:**

- `1 <= nums.length <= 104`
- `-1000 <= nums[i] <= 1000`


# Solution: prefix sum (running)

```python
class Solution:
    def pivotIndex(self, nums: List[int]) -> int:
        """
        Summary:
            calculate the pivot index of this array
            the pivot index is the index where the sum of all numbers
            strictly to the left of the index is equal to the sum of all the numbers strictly to the index's right.

            If the index is on the left edge of the array,
            then the left sum is 0 because there are no elements to the left. 

            This also applies to the right edge of the array. 

        Args:
            arr of integers `nums`
        Returns:
            the leftmost pivot index
            -1  if no such index exists
        """
        # brute force, just sum it out? 
        # for an index i, sum([0...i-1]) == sum([i+1, -1])
        # use prefix sum?
        # looks like format of it. 
        # sum([0...i-1]) is prefix sum of i-1
        # sum([i+1, -1]) is (prefix sum of -1) - prefix sum of i

        # first pass to construct prefix sum
        # prefix_sum = [0 for _ in range(len(nums))] # spatial O(N)
        # prefix_sum[0] = nums[0]
        # for i in range(1, len(nums)): # O(N)
        #     prefix_sum[i] = nums[i] + prefix_sum[i-1]
        sum_arr = sum(nums) # O(N)
        left_sum = 0
        for i in range(len(nums)):
            right_sum = sum_arr - (left_sum + nums[i])
            if left_sum == right_sum:
                return i
            left_sum += nums[i]
        return -1
```


Space complexity O(1) (can do it without constructing a prefix sum array. cause all we need is the full sum)

Time complexity O(N)
